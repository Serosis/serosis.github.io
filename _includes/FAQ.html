How do you install the mods?

Well first you make a backup copy of your original .exe, which should be by default in your C:\Program Files\The Ur-Quan Masters\ directory. The file mentioned is named uqm.exe, you can change the name to something else like uqm-backup.exe or uqm-original.exe. Don't worry if you can't see the .exe bit in the filename. By default that view is turned off.

After you have completed the above you can UnZIP the mod you downloaded straight into the directory aformentionted, which again by default is C:\Program Files\The Ur-Quan Masters\
When that is all done you can run the game.

What the heck is a .diff?

The .diff files are here only because I thought it would more efficient than to upload the entire source code for such relatively minor changes. These files should only be of use if you are planning to compile your own version of these mods or if you are cross compiling. To use these patch files on Windows you need to set up MinGW and MSYS, which is explained in the INSTALL.mingw file that comes with UQM's source code.

Copy the releveant .diff file you want into the /uqm-0.7.0/sc2 directory, open up a Terminal(MSYS for Windows users), cd into the directory where you placed the .diff and execute the command, patch -p0 -i *.diff. * Being the name of the .diff.
If all goes without a hitch you can begin compiling the source code.

How do I install the 3DO videos?

Windows

To make things easier on the community I have made available all the videos to download in one handy .zip file. You can get this file on the Downloads page.

To use them just extract the folder from within the zip file to your addons directory. Which by default is

C:\Program Files\The Ur-Quan Masters\content\addons\
or
C:\Program Files (x86)\The Ur-Quan Masters\content\addons\ in 64bit Windows

After that you can just load up UQM and opt to use the 3DO videos instead of the PC slides.

MacOS X

Right click on the UQM app and select Show Package Contents
Navigate to Contents -> Resources -> content -> addons
Extract the uqm-0.7.0-3dovideos.zip file into the addons folder.

How do I use the MegaMod cheats?

Windows

For Windows you can use the cheats through commandline, batch file, in the uqm.cfg file, or in a shortcut. Say you wanted to use a desktop shortcut, you would right-click the shortcut and go into its properties and add any cheat at the end of the "Target:". It should look like this.

I would advise against using the uqm.cfg method simply because the cheats are off by default and if you were to change any settings while in UQM it would reset the uqm.cfg file. In other words, get the settings how you want them first before applying cheats.

MacOS X

For OS X things get a bit more complicated. You can run commandline arguments using the open command, add them to the .cfg, or use the Automator script I included with the app.

Commandline

In this example I will assume you installed the MegaMod .app into the Applications folder:
For those that are already familiar with using the Terminal commandline, arguments using the open command should be a walk in the cake. Simply open up your Terminal and type in: open -a "The Ur-Quan Masters MegaMod.app" --args or open -a "The Ur-Quan Masters HD MegaMod.app" --args Depending on the version you installed. Then you type in whichever commandline argument you wish after. So if you wanted the headstart cheat you'd type in: --headstart After --args
You can put in as many commandline arguments you want after: --args

Editing the uqm.cfg

Word of warning. Changes to the .cfg file are reset every time you enter the setup menu in-game. So make whatever changes you need in-game before editing the .cfg

To edit the .cfg file you have to run UQM or UQM-HD at least once and go to their respective setup menus. This creates (and resets) the uqm.cfg file.
Open Finder, on the Menu Bar select: Go -> Go To Folder... And type in either: ~/.uqm/ or ~/.uqmhd/ Depending on the version you installed This brings you to the user folder that contains the uqm.cfg file.

Open the uqm.cfg file with any text editor you have. Inside there should be a nice unordered list of options which include all of the cheat options. Again I'll use headstart as an example. Find: headStart = BOOLEAN:false Change, "false" to, "true" and save the the changes.

Using the CMD Line Automator

This process assumes you installed UQM or UQM-HD in the Applications folder.

Copy CMD Line Automator wherever you'd like. Open Automator, you can search for it in Spotlight or open it up directly from your Applications folder. You will be greeted with a, "new file" window, at the bottom select Open an Existing Document... and select the CMD Line Automator file. It should look something like this.

From here on the process is the same as using the commandline options except that after you're done adding in the options you save the file and run it. You can put the CMD Line Automator anywhere you want, all it does is open up the app using the commandline options.
