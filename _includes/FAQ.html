Well first you make a backup copy of your original .exe, which should be by default in your C:\Program Files\The Ur-Quan Masters\ directory. The file mentioned is named uqm.exe, you can change the name to something else like uqm-backup.exe or uqm-original.exe. Don't worry if you can't see the .exe bit in the filename. By default that view is turned off.

After you have completed the above you can UnZIP the mod you downloaded straight into the directory aformentionted, which again by default is C:\Program Files\The Ur-Quan Masters\
When that is all done you can run the game.

The .diff files are here only because I thought it would more efficient than to upload the entire source code for such relatively minor changes. These files should only be of use if you are planning to compile your own version of these mods or if you are cross compiling. To use these patch files on Windows you need to set up MinGW and MSYS, which is explained in the INSTALL.mingw file that comes with UQM's source code.

Copy the releveant .diff file you want into the /uqm-0.7.0/sc2 directory, open up a Terminal(MSYS for Windows users), cd into the directory where you placed the .diff and execute the command, patch -p0 -i *.diff. * Being the name of the .diff.
If all goes without a hitch you can begin compiling the source code.

To make things easier on the community I have made available all the videos to download in one handy .zip file. You can get this file on the Downloads page.

To use them just extract the folder from within the zip file to your addons directory. Which by default is

C:\Program Files\The Ur-Quan Masters\content\addons\
or
C:\Program Files (x86)\The Ur-Quan Masters\content\addons\ in 64bit Windows

After that you can just load up UQM and opt to use the 3DO videos instead of the PC slides.

You can use the cheats through commandline, batch file, in the uqm.cfg file, or in a shortcut. Say you wanted to use a desktop shortcut, you would right-click the shortcut and go into its properties and add any cheat at the end of the "Target:". It should look like this.

I would advise against using the uqm.cfg method simply because the cheats are off by default and if you were to change any settings while in UQM it would reset the uqm.cfg file. In other words, get the settings how you want them first before applying cheats.
